# BetterTeam

**中文名：** 更好的队伍  
**Mod ID：** `better_team`

## 简介

BetterTeam 是一个适用于 **Fabric 1.21.10** 的服务端模组，用于对原版 `/team` 队伍系统进行小幅增强。

原版中，只有同队玩家之间默认无法互相攻击；但队伍内的其他实体仍然会受到同队玩家的伤害。  
例如，原版中红队玩家依然可以攻击加入红队的僵尸。

安装 BetterTeam 后，这一规则将扩展到所有加入同一原版队伍的实体：

- 同队实体之间无法互相造成伤害
- 同队来源的无差别攻击也会被正确拦截
- 同队发射的弹射物不会被同队实体阻挡，而是像原版同队玩家判定一样直接穿透

本模组仅作用于原版 `/team` 队伍系统，不提供新的队伍系统，也不包含额外配置项。

---

## 功能特性

### 1. 扩展原版“同队免伤”判定到所有实体

原版的同队免伤主要体现在玩家之间。  
BetterTeam 会将这套逻辑扩展到所有加入同一队伍的实体，使它们无法受到来自同队实体的伤害。

这意味着以下情况都会被阻止：

- 同队玩家攻击同队生物
- 同队生物攻击同队玩家
- 同队生物攻击同队生物

### 2. 处理多种伤害来源

BetterTeam 不仅处理普通近战，还会处理来自同队实体的其他伤害来源，包括但不限于：

- 近战攻击
- 弓箭、弩箭、三叉戟、火球等投射物
- 河豚
- 闪电
- 唤魔者尖牙
- TNT
- 末影水晶爆炸
- 药水、滞留药水
- 区域效果云
- 驯服生物替主人造成的伤害

### 3. 同队弹射物可穿透同队实体

同队发射的弹射物不会被同队实体阻挡。

例如：

- 红队玩家射出的箭会穿过红队实体
- 该箭仍可正常命中非红队实体

这样可以让同队协作更加自然，也更接近原版同队玩家之间的相关判定体验。

### 4. 不影响环境伤害

BetterTeam 只处理来自同队实体的伤害。  
环境伤害不受影响，例如：

- 掉落伤害
- 火焰或岩浆伤害
- 窒息伤害
- 虚空伤害
- 其他非同队实体直接造成的环境类伤害

---

## 适用场景

BetterTeam 适合用于以下类型的服务器：

- 小游戏服务器
- PVE / RPG 服务器
- 阵营对抗服务器
- 需要让召唤物、宠物或自定义队伍实体真正共享队伍保护的玩法环境

尤其适合需要让“玩家、宠物、召唤物、怪物单位”统一纳入原版队伍判定的场景。

---

## 运行环境

- **Minecraft：** `1.21.10`
- **Mod Loader：** Fabric
- **Fabric Loader：** `0.18.1`
- **Fabric API：** 不需要
- **类型：** 服务端模组

客户端通常无需安装本模组。

---

## 安装方法

1. 安装对应版本的 Fabric Loader。
2. 将 BetterTeam 模组文件放入服务端的 `mods` 文件夹。
3. 启动服务器。

---

## 使用方式

本模组不添加新命令，直接增强原版 `/team` 系统。

你仍然使用原版命令来管理队伍，例如：

```mcfunction
/team add red
/team join red PlayerName
/team join red @e[type=zombie,limit=1,sort=nearest]
